## Description
Participation au challenge de programmation de 30 jours organisé par hackacademy-lab.

## Objectif
Résoudre un défi de programmation chaque jour pendant 30 jours et améliorer mes compétences en développement.

## Structure du projet
Chaque dossier JourX contient :
- Un fichier README.md décrivant la solution
- Le code source correspondant
